# spectrum-brio
Health application for the Spectrum Community


# UI Setup
https://github.com/react-boilerplate/react-boilerplate

Terminal Commands To Start UI Locally (After Pulling From Git)

CD brio-front-end

npm run setup

npm start

by default will run on http://localhost:3000/
